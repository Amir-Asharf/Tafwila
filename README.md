Tafwela - Roadside Assistance App
Tafwela is a Flutter-based mobile application designed to assist drivers in case of car breakdowns on the road. Users can open the app, access the map to pinpoint their location, and communicate with nearby fuel stations through a chatbot. The fuel station dispatches a vehicle to the user's location with the requested service. The app is built using Flutter and integrates APIs for backend services, including real-time communication and location tracking. This is my graduation project, showcasing advanced API integration with Flutter.
